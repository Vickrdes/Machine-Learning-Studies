# 🤖 Machine Learning Studies

Repositório dedicado ao estudo e implementação de algoritmos de **Aprendizado de Máquina**, com foco em rigor técnico, análise estatística e reprodutibilidade.

## 🚀 Competências em Desenvolvimento

Atualmente, estou aperfeiçoando minhas habilidades nas seguintes áreas do fluxo de Ciência de Dados:

* **🔍 EDA (Exploratory Data Analysis):** Realizo a exploração profunda de datasets para identificar padrões, correlações e a presença de outliers, utilizando **cálculos estatísticos** para validar hipóteses sobre os dados.
* **⚙️ Pré-processamento:** Tratamento de dados faltantes, normalização e escalonamento (StandardScaler), garantindo que os dados estejam prontos para algoritmos sensíveis à escala.
* **🧠 Modelagem SVM (Support Vector Machine):** Estudo de fronteiras de decisão, comparação de kernels (Linear vs RBF) e ajuste de hiperparâmetros (C e Gamma) para otimizar o equilíbrio entre Bias e Variance.

## 🛠️ Tecnologias e Bibliotecas
Para o desenvolvimento desses estudos, utilizo o ecossistema Python focado em dados:
- **Pandas & NumPy:** Manipulação de matrizes, vetores e estruturação de DataFrames.
- **Scikit-learn:** Implementação de modelos de ML e métricas de avaliação.
- **Matplotlib & Seaborn:** Visualização de dados e matrizes de confusão.

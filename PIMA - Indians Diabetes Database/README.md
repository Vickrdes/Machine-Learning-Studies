# 🩺 PIMA Indians Diabetes Database
### Desafio: Previsão de Diabetes em Pacientes

Este estudo foca na análise de fatores de risco (como glicose, IMC e idade) para prever a ocorrência de diabetes em mulheres da herança Pima Indian.

---

## 🎯 Objetivo
Desenvolver um modelo capaz de identificar pacientes com alto risco de diabetes, lidando com os desafios de dados faltantes (valores zero em colunas críticas).

## 📊 Estratégia de Análise
* **Data Cleaning:** Identificação e tratamento de valores inconsistentes em variáveis como Insulina e Pressão Arterial.
* **Análise Exploratória:** Visualização da correlação entre as variáveis fisiológicas e o desfecho positivo para diabetes.
* **Escalonamento:** Preparação dos dados para garantir que características com escalas maiores (como Glicose) não dominem o modelo.

---
> **Dataset Original:** [Kaggle - PIMA Indians Diabetes](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)
